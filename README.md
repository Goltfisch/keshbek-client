# keshbek-client

## Project setup
```
npm install
```

Create .env.development File with
```
VUE_APP_BASE_URI="http://localhost:8000/api"
```
for local development


### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
